# Audio library (the engine auto-uses these)

Drop royalty-free files here (Pixabay Music, Uppbeat, Freesound). The render
job maps cue types -> filenames, so keep these names (add variants as -2, -3):

## sfx/  (fired at cuts/transitions/emphasis)
- whoosh.mp3     -> scene cut / card in
- click.mp3      -> small UI tick / word emphasis
- pop.mp3        -> element appear
- riser.mp3      -> build into a reveal
- keyboard.mp3   -> typing / code moments
- thud.mp3       -> hard stop / punchline
